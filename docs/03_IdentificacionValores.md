

# Identificación de Valores a Imputar {#Identificacion}

## Reglas de Validación

Rango de valores válidos. Validación de dominios. Consistencia inter-variable. Coherencia lógica (ejemplo: edad vs. nivel educativo).

## Clasificación de Casos

Valores faltantes (NA), Valores fuera de rango, Inconsistencias lógicas, Registros duplicados (si aplica).

## Cuantificación

Porcentaje de no respuesta por variable, Porcentaje por dominio geográfico, Tablas de frecuencia, Mapas de calor o matrices de completitud, etc.
